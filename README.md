# 월클플레이 GitHub Pages 사이트 패키지

## 적용 방법

1. 이 ZIP 파일을 압축 해제합니다.
2. GitHub 저장소 루트에 모든 파일과 폴더를 업로드합니다.
3. 기존 `index.html`은 새 파일로 교체합니다.
4. GitHub Pages 주소가 `https://cardposman.github.io/` 로 열리는지 확인합니다.
5. 네이버 서치어드바이저에 사이트를 등록합니다.
6. `robots.txt`와 `sitemap.xml`을 제출합니다.

## 포함된 주요 파일

- `index.html`

## 중요

외부 공식 신청 링크는 `<a href>`가 아니라 `button onclick` 방식으로 처리했습니다.
내부 페이지 이동 링크는 검색엔진 수집을 위해 `<a href>` 방식으로 유지했습니다.
